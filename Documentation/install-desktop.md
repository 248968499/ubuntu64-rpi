# 安装桌面方法

### 这些桌面环境都是开箱即用的

1，mate桌面环境

`apt-get install ubuntu-mate-desktop`

2，gnome桌面环境

`apt-get install gnome`

3，lxde桌面环境

`apt-get install lxde`

4，lxqt桌面环境

`apt-get install lxqt`

5，xubuntu桌面

`apt-get install xubuntu-desktop`

6，lubuntu桌面

`apt-get install lubuntu-desktop`

7，xfce桌面环境

`apt-get install xfce4`
